# Jetson and ESP32 Code

Work-in-progress repository for small ESP32 experiments and future Jetson integration.

For now, this repo includes early Arduino/ESP32 sketches for:
- ADC voltage reading
- Knock detection using an accelerometer
- Buzzer, LEDs, and LCD feedback

The broader goal is to build embedded projects that combine ESP32 boards, sensors, cameras, and an NVIDIA Jetson for real-time sensing, computer vision, and event logging.

This repository is experimental and will grow as the projects become more complete.
